AI Bayan 2025 — Online GitHub Edition (v6, без музыки)
=====================================================

Как опубликовать на GitHub Pages:
1) Создайте репозиторий: AI_Bayan_2025 (Public).
2) Загрузите все файлы из этой папки (index.html, assets, public, src → после сборки при необходимости).
   Этот архив уже работает как статический сайт (Vite dev/preview для локального запуска).
3) Включите Pages: Settings → Pages → Deploy from branch → main → /(root) → Save.
4) Откройте ссылку вида: https://<ваш_логин>.github.io/AI_Bayan_2025/
